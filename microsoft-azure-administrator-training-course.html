AZ-104 Exam & Certification

To earn the Azure Developer certification, you must pass the following exam:
AZ-104: Microsoft Azure Administrator
NOTE: This exam is a replacement of AZ-103: Microsoft Azure Administrator which got updated to AZ-104 on APRIL 2, 2020. You will be able to take the old exam until it retires on August 31, 2020.

Yes, this Microsoft Certified Azure Administrator Associate AZ-104 is the replacement for Exam AZ-103. This holistic curriculum prepares you for the exam.

This new Microsoft Certified: Azure Administrator Associate certification has evolved to have more in-depth coverage of the role-specific skills required in the industry compared to the previous certification.

Yes, Trident Global Services is a Silver Microsoft Partner, and this course is accredited by Microsoft.

Yes, the exam fee is included in the course fee.

The cost of the Exam AZ-104: Microsoft Azure Administrator is $165.00 USD plus taxes (for US residents) each and INR 4800 plus taxes (for India residents) each. This fee is paid to Microsoft when you register for your exam.
